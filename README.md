## Installation

1. Clone the repository

```bash
git clone https://github.com/shaurya713/portfolio.git
```

2. Navigate to the project folder

```bash
cd portfolio
```

3. Open the project

```bash
open index.html
```

or simply double-click the `index.html` file.
